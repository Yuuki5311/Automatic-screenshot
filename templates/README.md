# 模板图准备指南

运行脚本前，需要在 `templates/` 目录下放入以下按钮模板图。
模板图通过首次手动进入游戏后截取小图获得。

## 截取方法

1. 运行脚本，在 QQ 登录完成后手动进入对应页面
2. 截取整个游戏画面
3. 用小画图工具裁剪出目标按钮区域（建议裁剪 30-50px 边距）
4. 保存为对应的文件名
